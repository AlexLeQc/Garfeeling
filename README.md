# Garfeeling

![Garfeeling](frontend/public/images/logo.png)

## Description

Garfeeling is a web application that allows you to trigger sounds on a remote device using Socket.IO. It's a fun little project that can be used for pranks or just to have some fun with friends.

## Tech Stack

- **Frontend**: Angular 21
- **Backend**: Node.js
- **Real-time communication**: Socket.IO

## Getting Started

### Prerequisites

- Node.js >= 14.0.0
- Angular CLI >= 16.0.0

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/AlexLeQc/Garfeeling.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```

## Usage

### Frontend

Open [http://localhost:4200](http://localhost:4200) in your browser to access the application.

### Backend

Open [http://localhost:3000](http://localhost:3000) in your browser to access the API documentation.
